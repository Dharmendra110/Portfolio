# Professional Portfolio

A modern, responsive portfolio website showcasing professional experience, education, projects, and technical skills. Built with React and Vite for optimal performance and fast development.

## 🎯 Features

- **Home** - Engaging landing page with introduction
- **About** - Comprehensive background and professional summary
- **Experience** - Detailed work history and achievements
- **Education** - Academic qualifications and certifications
- **Skills** - Technical expertise and proficiencies
- **Projects** - Portfolio of completed work and personal projects
- **Resume** - Downloadable resume/CV
- **Contact** - Easy ways to get in touch
- **Responsive Design** - Fully mobile-optimized interface

## 🚀 Technologies

- **React** - UI library
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **PostCSS** - CSS transformation tool
- **ESLint** - Code quality and consistency

## 📋 Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd portfolio
```

2. Install dependencies:

```bash
npm install
```

## 🏃 Running Locally

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## 🔨 Building for Production

Create an optimized production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
